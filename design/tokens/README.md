# Design tokens – No regime for me

`colors.md`, `spacing.md`, and `typography.md` are the **source of truth** for this plugin’s admin UI. They align with WordPress admin standards so the plugin feels native to wp-admin.

- **Colors:** Semantic roles mapped to the WordPress 5.7+ admin palette (WCAG 2.0 AA).
- **Spacing:** 4px-based scale with usage guidance for forms, sections, and layout.
- **Typography:** Font stack and type roles matching wp-admin (page title 24px, body 14px, etc.).

The canonical tokens are `colors.md`, `spacing.md`, and `typography.md` above.
