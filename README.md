# No Regime For Me

Uses WordPress admin spaces to educate and remind on the dangers of fascism and authoritarianism. Short quotes and tips in the admin bar; longer content in a dashboard widget or via the shortcode `[no_regime_content]`. Settings: **No Regime For Me** in the admin menu.

**Requirements:** WordPress 5.9+, PHP 7.4+

**Install:** Place the plugin folder in `wp-content/plugins/`, activate. Or zip the folder and use **Plugins → Add New → Upload Plugin**.
